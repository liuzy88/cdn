# 文件上传客户端

## 用途
- 命令行上传文件到七牛存储，支持linux, mac, windows

## 试用
- 你可以从我的CDN下载此程序
- [下载Linux x64](http://upload.liuzy88.com/cli-linux)
- [下载Mac x64](http://upload.liuzy88.com/cli-macos)
- [下载Windows x64](http://upload.liuzy88.com/cli-win.exe)
